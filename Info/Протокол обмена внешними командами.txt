Протокол обмена внешними командами между базами (APTEKA_NET) различных серверов
-------------------------------------------------------------------------------

  - Формат команды:

                        <имя команды><параметры команды>, 

  где <имя команды> одно из множества допустимых (длина имени команды 5 символов);
  <параметры команды> параметры индивидуальные для каждой команды, длина не более 250 символов.

  Попутно с командой, через интерфейс выполняемой программы, передаються дополнительные параметры 
Owner (имя приложения, которое будет выполнять команду). Команда регистрируется в базе данных SQL 
в таблице COMMAND (по умолчанию)
