---------------------------------------------------------------------------------
   				История	изменения версий.
---------------------------------------------------------------------------------
Версия 1.1.9.27	   -	Добавленено API для работы с графическими документами 
			(купонами). Описание API есть в документации на 
			соответствующие версии, в которых это API поддерживается.

Версия 1.1.10.28   - 	Замена абревиатуры "Держатель ПК" на "Держатель ЕПЗ" при 
			оплате покупки пластиковой банковской картой согласно 
			Приказу Министерства Финансов Украины №13 от 21.01.2016.

Версия 1.1.11.29   - 	Добавлены в каждый объект следующие свойства:

			1. glVirtualPortOpened (чтение) - обеспечивает контроль
			 состояния соединения с ФР через виртуальный COM-порт
			 при подключении через USB-интерфейс.

			2. glUseVirtualPort (чтение/запись) - используется в случае
			 подключения ФР по интерфейсу USB и корректной работы с
			 виртуальным COM-портом в дальнейшем. Это свойство однозначно
			 нужно использовать для ФР IKC-E07, т.к. у него нет других 
			 интерфейсов, кроме USB.

Версия 1.1.12.38   - 	1. Добавлена возможность управления ФР по USB-интерфейсу через WinUSB драйвер.
			2. В последних версиях ФР, за исключением ФР IKC-E07 добавлена возможность 
			 управления по интерфейсу Ethernet. Для управления используется TCP протокол.
			 Кроме этого добавлен поиск ФР в сети Ethernet по протоколу UDP.
			3. Добавлены новые функции в графический документ. (см. документацию).
